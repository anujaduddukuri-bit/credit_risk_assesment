# 💳 Credit Risk Assessment using Machine Learning

A Machine Learning-based web application that predicts whether a loan applicant is a **Good Credit Risk** or **Bad Credit Risk** based on financial and personal information. This project demonstrates the complete ML workflow, from data preprocessing to model deployment.

---

## 📌 Overview

Credit risk assessment is an essential task for financial institutions to minimize loan defaults. This project uses machine learning algorithms to analyze applicant data and predict creditworthiness, helping support better lending decisions.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/anujaduddukuri-bit/credit_risk_assesment.git
```

Navigate to the project directory:

```bash
cd credit_risk_assesment
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the Flask application:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
